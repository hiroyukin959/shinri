# 現在の関心事

## 人の厄介な特性

### うっぷんばらし

ストレスから人にあたる。あたり方は人それぞれ。

悪意あるいたずらもこの範疇か。

### 仕返し

自身にとって不快なことをされた場合の行動の一種。

不快と感じる内容は人それぞれ。だから、いつ仕返しされるかわからない。

## アホをのさばらせない

アホのいない世界で暮らすのが一番。

悪い行動を指摘する。
